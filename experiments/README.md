# Experiments

This folder contains experimental notebooks used during development and testing.

## Contents

- **test_api.ipynb** - API testing and validation
- **open_source_models.ipynb** - Experiments with open-source models

These notebooks are kept for reference and development purposes. They may contain:

- Work-in-progress features
- API testing code
- Model comparisons
- Performance benchmarks
- Development notes

## Note

For production-ready applications, use the notebooks in the `app/` folder instead.
